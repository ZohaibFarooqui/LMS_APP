# LMS Mobile App

Cross-platform Flutter application for Yousuf Dewan Companies' Attendance & Leave Management System built with Flutter Clean Architecture and feature-first BLoC modules.

## Documentation
- [Business Requirements Document](docs/BRD.md)

## Getting Started
1. Install Flutter (3.9.x) and run `flutter pub get`.
2. Launch the app with `flutter run` or execute unit tests with `flutter test`.
3. Refer to the BRD for detailed scope, architecture, and testing strategy (SPRITE matrix).
